# Snabbit Agent Console

The Snabbit Agent Console is an internal AI workflow console for Snabbit's ops
team — a dense, dark, Linear-grade dashboard for running SDLC agents such as PR
review, deploys, RCAs, and alert triage. It is backed by a REST API and a live
CI/CD integration.

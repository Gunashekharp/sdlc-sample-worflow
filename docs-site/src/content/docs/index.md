The Snabbit Agent Console is an internal AI workflow console for Snabbit's ops
team — a dashboard for running SDLC agents such as PR review, deploys and alert
triage, backed by a REST API and a live CI/CD integration.

:::note
This documentation is maintained automatically. On a schedule, a documentation
agent reviews the codebase and updates these pages.
:::

## Where to start

Use the sidebar to browse the documentation. On its first run, the agent
expands this into a complete reference for the frontend, the backend, and the
REST API.
